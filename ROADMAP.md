## Capstone Project (Days 88–90)

Build a **Task Manager API** using everything learned:
- FastAPI + PostgreSQL + SQLAlchemy
- JWT auth (register/login/protected routes)
- Full CRUD for tasks (create, list, update, delete, filter by status)
- Background task for email notification on assignment
- 80%+ test coverage with pytest
- Dockerized and deployed
